## 003_escolaridad

Esta libreta solo se enfoca en el tratamiento previo de los datos,una vez ejecutada genera un archivo con los datos necesarios, esos datos ya estan en la carpeta data. Para ejecutar la libreta `003_escolaridad` es necesario descargar los archivos de los censos:

* [Censo poblacional 2020](https://www.inegi.org.mx/contenidos/programas/ccpv/2020/datosabiertos/iter/iter_00_cpv2020_csv.zip)
* [Censo poblacional 2010](https://www.inegi.org.mx/contenidos/programas/ccpv/2010/datosabiertos/iter_nal_2010_csv.zip)
* [Censo poblacional 2005](https://www.inegi.org.mx/contenidos/programas/ccpv/2005/datosabiertos/cpv2005_iter_00_csv.zip)
* [Censo poblacional 2000](https://www.inegi.org.mx/contenidos/programas/ccpv/2005/datosabiertos/cpv2005_iter_00_csv.zip)

Posteriormente se deberan extraer en la carpeta ´../data/escolaridad/´. Una vez hecho esto se puede ejecutar el cuaderno sin errores

 Los censos poblacionales contemplan mas de 150 campos y registran todas las localidades del pais. Se **extrajeron solamente** datos del nivel de escolaridad por estado del pais y promedio nacional.
